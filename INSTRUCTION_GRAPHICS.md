# Benötigte Abstrakte Anleitungs-Grafiken (Abstract Instruction Graphics)

Basierend auf der Analyse der vorhandenen Rezepte (Backen, Kochen, Frühstück, etc.) wurde diese Liste erstellt. Ziel ist es, Grafiken zu erstellen, die allgemein genug sind, um wiederverwendet zu werden, aber spezifisch genug, um den Arbeitsschritt klar zu kommunizieren.

## 1. Vorbereitung (Preparation)

Diese Schritte kommen fast überall vor.

| Name / Aktion | Beschreibung für das Design | Beispiele im Rezept |
| :--- | :--- | :--- |
| **Zutaten schneiden** | Ein Messer auf einem Schneidebrett, daneben angedeutetes Gemüse/Obst (abstrakt). | "Äpfel würfeln", "Zwiebel hacken", "Hähnchen schneiden" |
| **Zutaten schälen** | Ein Sparschäler, der eine Schale (z.B. von Kartoffel/Apfel) abzieht. | "Äpfel schälen", "Kartoffeln schälen" |
| **Reiben** | Eine Küchenreibe mit etwas Geriebenem davor. | "Ingwer reiben", "Parmesan reiben", "Zitronenschale" |
| **Mischen / Verrühren** | Eine Rührschüssel mit einem Schneebesen oder Löffel darin. | "Eier verquirlen", "Zutaten mischen", "Dressing anrühren" |
| **Abmessen** | Ein Messbecher oder eine Küchenwaage. | "Zutaten abmessen", "Mehl wiegen" |
| **Waschen** | Hände oder Gemüse unter einem Wasserhahn/Wasserstrahl. | "Gemüse waschen", "Salat waschen" |

## 2. Kochen am Herd (Stovetop Cooking)

| Name / Aktion | Beschreibung für das Design | Beispiele im Rezept |
| :--- | :--- | :--- |
| **Pfanne erhitzen / Braten** | Eine Pfanne auf einer Herdplatte (eventuell mit kleinem Öltropfen oder Flamme drunter). | "Öl erhitzen", "Speck braten", "Pancakes backen" |
| **Kochen (Wasser/Suppe)** | Ein Topf auf dem Herd mit blubberndem Wasser/Inhalt (Dampf aufsteigend). | "Pasta kochen", "Suppe kochen", "Kartoffeln kochen" |
| **Köcheln lassen** | Ein Topf mit Deckel drauf, leichter Dampf (Zeigt "Warten" / "Langsam"). | "Köcheln lassen", "Reis garen" |
| **Hinzufügen (in Topf/Pfanne)** | Eine Hand oder Schüssel, die etwas in einen Topf/Pfanne kippt. | "Sahne hinzufügen", "Nudeln abgießen und zurückgeben" |
| **Umrühren (Topf/Pfanne)** | Kochlöffel in einem Topf/Pfanne (Kreisbewegung angedeutet). | "Stetig rühren", "Umrühren" |

## 3. Backen (Baking)

| Name / Aktion | Beschreibung für das Design | Beispiele im Rezept |
| :--- | :--- | :--- |
| **Ofen vorheizen** | Ein Ofen-Symbol oder Backofenfront mit einer Temperaturanzeige. | "Ofen auf 180°C vorheizen" |
| **Teig kneten** | Hände, die einen Teigballen bearbeiten. | "Zu Streuseln verkneten", "Hefeteig kneten" |
| **Ausrollen** | Ein Nudelholz auf einer Fläche (evtl. mit Teigplatte). | "Teig ausrollen", "Pizza formen" |
| **In Form geben** | Eine Backform (Kastenform oder rund) oder ein Backblech. | "In Auflaufform geben", "Auf Blech legen" |
| **Backen (Zeit)** | Ein Ofen mit geschlossener Tür und einer Uhr/Sanduhr daneben. | "30 Min backen", "Goldbraun backen" |
| **Abkühlen lassen** | Ein Kuchengitter oder Teller mit dampfendem Essen darauf. | "Abkühlen lassen", "Ruhen lassen" |
| **Trennen (Eier)** | Zwei Eierschalen-Hälften, Eigelb wird getrennt. | "Eier trennen" |

## 4. Geräte & Spezial (Equipment)

| Name / Aktion | Beschreibung für das Design | Beispiele im Rezept |
| :--- | :--- | :--- |
| **Mixen / Pürieren** | Ein Standmixer (Blender) oder Pürierstab. | "Smoothie mixen", "Suppe pürieren" |
| **Handrührgerät** | Ein elektrischer Handmixer. | "Sahne steif schlagen", "Teig rühren" |
| **Sieben** | Ein Sieb, durch das Pulver (Mehl/Puderz) rieselt. | "Mehl sieben", "Mit Puderz bestäuben" |

## 5. Anrichten & Servieren (Finishing)

| Name / Aktion | Beschreibung für das Design | Beispiele im Rezept |
| :--- | :--- | :--- |
| **Servieren** | Ein fertig angerichteter Teller mit Besteck daneben. | "Servieren", "Anrichten" |
| **Garnieren / Topping** | Eine Hand, die etwas (Kräuter/Nüsse) über eine Schüssel/Teller streut. | "Mit Petersilie garnieren", "Toppings hinzufügen" |
| **Würzen / Abschmecken** | Salz- und Pfefferstreuer. | "Abschmecken", "Salzen und pfeffern" |

## Zusammenfassung für das Design

Für ein konsistentes Set sollten Sie sich auf **ca. 20-25 Icons/Grafiken** konzentrieren, die 90% der Fälle abdecken:

1.  **Schneiden** (Messer)
2.  **Schälen** (Schäler)
3.  **Reiben** (Reibe)
4.  **Mischen** (Schüssel + Besen)
5.  **Kneten** (Hände + Teig)
6.  **Waage/Messen**
7.  **Pfanne (Braten)**
8.  **Topf (Kochen)**
9.  **Topf (Köcheln/Deckel)**
10. **Ofen (Vorheizen)**
11. **Ofen (Backen/Timer)**
12. **Backform/Blech**
13. **Nudelholz**
14. **Mixer (Blender)**
15. **Handmixer**
16. **Sieb**
17. **Hinzufügen/Eingießen**
18. **Umrühren (am Herd)**
19. **Ei trennen**
20. **Servieren (Teller)**
21. **Garnieren/Streuen**
22. **Würzen (Salz/Pfeffer)**
