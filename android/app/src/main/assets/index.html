Daily Dish Decider

Dies ist das minimalistische Android-WebView-Shellprojekt. Die Web-App wird lokal aus assets/index.html geladen.

Führe npm run build aus und kopiere das erzeugte dist/ Verzeichnis nach android/app/src/main/assets/, um die aktuelle Weboberfläche im APK zu bündeln.

Standardmäßig zeigt der Build diese Platzhalterseite an.
